### BR-TR-TAX-014: Progressive Tax Calculation

**Priority**: HIGH

**Description**:
Calculate tax using progressive brackets.

**Conditions**:
```
FOR each bracket WHERE taxable_income > bracket_min
  tax += (min(taxable_income, bracket_max) - bracket_min) × rate
```

**Rules**:
1. Apply each bracket rate to income in that bracket
2. Sum tax from all applicable brackets
3. Alternative: Use quick deduction formula
4. Formula: (taxable_income × rate) - quick_deduction
5. Round to nearest whole currency unit

**Exceptions**:
- None

**Error Messages**:
- `ERR_TAX_014`: "Progressive tax calculation failed: {reason}."

**Related Requirements**: FR-TR-TAX-014

### BR-TR-TAX-017: FICA Tax Calculation (US)

**Priority**: HIGH

**Description**:
Calculate FICA taxes (Social Security + Medicare).

**Conditions**:
```
social_security = min(gross, SS_cap) × 6.2%
medicare = gross × 1.45%
additional_medicare = max(0, gross - 200K) × 0.9%
```

**Rules**:
1. Social Security: 6.2% up to wage cap ($168,600 in 2025)
2. Medicare: 1.45% on all wages (no cap)
3. Additional Medicare: 0.9% on wages > $200K
4. Employer matches Social Security and Medicare (not additional)
5. Year-to-date tracking for caps

**Exceptions**:
- None

**Error Messages**:
- None

**Related Requirements**: FR-TR-TAX-017

**Related Entities**: FICATax, TaxCalculation

### BR-TR-TAX-018: Unemployment Tax Calculation

**Priority**: MEDIUM

**Description**:
Calculate unemployment taxes (FUTA, SUTA).

**Conditions**:

4. Year-to-date tracking for caps
