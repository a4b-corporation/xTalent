# Payroll Glossary

> **Domain**: Payroll (PR)

---

## Core Terms

| Term | Definition | Vietnamese | Example |
|------|------------|------------|---------|
| **Pay Frequency** | Chu kỳ trả lương chuẩn | Tần suất trả lương | MONTHLY, BIWEEKLY |
| **Pay Calendar** | Lịch trả lương với cut-off dates | Lịch lương | VN-MONTHLY-2025 |
| **Pay Group** | Nhóm employees có cùng đặc điểm payroll | Nhóm trả lương | VN-HQ-STAFF |
| **Pay Profile** | Bundle của policies, rules, elements | Profile lương | VN-STANDARD |
| **Pay Element** | Thành phần lương (earning/deduction/tax) | Thành phần lương | BASIC_SALARY, OT_150 |
| **Pay Formula** | Công thức tính dùng chung | Công thức | BHXH_CALC_VN |
| **Balance Definition** | Định nghĩa balance (GROSS, NET, YTD) | Balance | YTD_GROSS |

---

## Classification Terms

| Term | Definition |
|------|------------|
| **EARNING** | Thu nhập - contributes positive to gross |
| **DEDUCTION** | Khấu trừ - reduces net pay |
| **TAX** | Thuế thu nhập cá nhân |
| **EMPLOYER_CONTRIBUTION** | Đóng góp của employer (không trừ lương) |
| **INFORMATIONAL** | Chỉ hiển thị, không calculation |

---

## Balance Types

| Type | Scope | Reset |
|------|-------|-------|
| **RUN** | Per payroll run | Each run |
| **PTD** | Period-to-date | Each period |
| **QTD** | Quarter-to-date | Quarterly |
| **YTD** | Year-to-date | Yearly |
| **LTD** | Lifetime-to-date | Never |
