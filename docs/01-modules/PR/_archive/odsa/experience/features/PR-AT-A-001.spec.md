# Feature Spec: Audit Trail Query

> **Feature ID**: PR-AT-A-001
> **Classification**: Analytics (A)
> **Priority**: P0 (MVP)
> **Spec Depth**: Medium
> **Date**: 2026-03-31

---

## Overview

Audit Trail Query enables HR Managers and Compliance Officers to search and view configuration change history. All Payroll module configuration changes are logged with complete audit trail including old/new values, change reason, and user attribution.

---

## Data Sources

| Source | Table | Description |
|--------|-------|-------------|
| Audit Log | pr_audit_log | Log container per scope |
| Audit Entry | pr_audit_entry | Individual change entries (immutable) |

**Audit Entry Fields**:

| Field | Type | Description |
|-------|------|-------------|
| entityType | Enum | PayElement, PayProfile, StatutoryRule, etc. |
| entityId | String | Entity identifier |
| entityName | String | Entity display name |
| operation | Enum | CREATE, UPDATE, DELETE, ASSIGN, VERSION_CREATE |
| changedBy | String | User who made change |
| changedByName | String | User display name |
| changedAt | Timestamp | When change occurred |
| oldValue | JSON | Previous state (for UPDATE) |
| newValue | JSON | New state |
| changedFields | Array | List of changed field names |
| changeReason | String | Reason for change |
| versionId | String | Link to SCD-2 version (if applicable) |

## Query Examples

### Example 1: Who changed BHXH rate last month?

| Filter | Value |
|--------|-------|
| dateRangeStart | 2026-02-01 |
| dateRangeEnd | 2026-02-28 |
| entityType | StatutoryRule |
| entityId | BHXH_EE |

**Result**: List of audit entries showing rate changes with user attribution.

### Example 2: All profile changes by admin@company

| Filter | Value |
|--------|-------|
| dateRangeStart | 2026-01-01 |
| dateRangeEnd | Today |
| entityType | PayProfile |
| userId | admin@company |

**Result**: All profile changes made by specific user.

## API Endpoints

| Endpoint | Method | Purpose |
|----------|--------|---------|
| /audit-logs | GET | Query audit entries |
| /audit-logs/{id} | GET | Get specific audit log |
| /audit-logs/{id}/entries | GET | Get entries for audit log |
| /audit-logs/entries/{entryId} | GET | Get specific entry detail |
| /audit-logs/export | GET | Export as CSV or PDF |

**Query Parameters**:

```
GET /audit-logs?dateRangeStart=2026-01-01&dateRangeEnd=2026-03-31&entityType=PayElement&page=1&limit=20
```
