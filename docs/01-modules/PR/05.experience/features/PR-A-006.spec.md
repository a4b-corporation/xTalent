# PR-A-006: Calculation Log Viewer

**Type**: A — Analytics
**Priority**: P0
**BC**: BC-03 Payroll Execution
**Actors**: PAYROLL_ADMIN
**Linked Stories**: US-046
**API Endpoints**: `GET /payroll/results/{resultId}/calc-log`

---

## Overview

Step-by-step calculation audit log for a specific worker × period payroll result. Enables Payroll Admin to trace exactly how gross, SI, PIT, and net were computed — every formula applied, every input variable used, and the output value at each step.

Access path: Payroll Run detail → Worker result row → "View Calc Log" link.

---

## Data Sources

| Source Table | Usage |
|-------------|-------|
| `calc_log_entry` | Ordered step list: element_code, formula_version, input_variables_json, output_value |
| `payroll_result` | gross_amount, si_employee_total, pit_amount, net_amount (summary header) |
| `pay_element` | element_name, element_type for display labels |

---

## Layout

### Header Section

Worker name + ID, period, pay group, run ID, result status (DRAFT / FINAL), result integrity hash (if FINAL).

### Gross-to-Net Pipeline Summary

Visual pipeline showing key totals:
```
[Gross: X,XXX,XXX] → [− SI: XXX,XXX] → [− PIT: XXX,XXX] → [− Other: XXX,XXX] → [Net: X,XXX,XXX]
```

Each segment is clickable — jumps to relevant steps in the log table.

### Calculation Steps Table

Paginated (50 rows per page), ordered by `step_order` ASC.

| Column | Description |
|--------|-------------|
| Step # | step_order |
| Element | element_code + element_name |
| Element Type | EARNING / DEDUCTION / SI / PIT / NET |
| Formula Version | formula_version string (e.g., "2024-01-01-v2") |
| Input Variables | Collapsed JSON; expand to see key=value pairs |
| Output Value | VND amount (formatted with thousand separators) |
| Notes | Optional calculation note |

### Colour Coding

- EARNING rows: green background (subtle)
- SI/PIT/DEDUCTION rows: amber background
- NET row: blue background, bold font

---

## Filter

- **Element Type**: filter to show only EARNING / SI / PIT / DEDUCTION / NET
- **Element Code**: text search to find a specific element across all steps
- **Formula Version**: filter to see all steps using a specific formula version

---

## Input Variables Expand

Clicking the Input Variables cell expands to show a key=value table:

| Variable | Value |
|----------|-------|
| base_salary | 15,000,000 |
| actual_work_days | 22 |
| standard_work_days | 26 |
| proration_ratio | 0.846 |
| formula | base_salary × proration_ratio |

---

## Export

- **PDF**: "Export as PDF" — generates full calc log as formatted PDF for audit evidence submission
- Includes: header, pipeline summary, full step table with all expanded input variables

---

## Performance Note

A single PayrollResult calc_log_entry can have 50–150 rows (depending on number of pay elements). Pagination (50 rows/page) is mandatory. Search/filter must use indexed queries on `(result_id, element_code, step_order)`.
