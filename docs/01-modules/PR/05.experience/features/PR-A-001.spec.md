# PR-A-001: Payroll Cost Report

**Type**: A — Analytics
**Priority**: P0
**BC**: BC-03 Payroll Execution
**Actors**: FINANCE_MANAGER, HR_MANAGER, CE_VIEWER (read-only cross-entity)
**Linked Stories**: US-036
**API Endpoints**: `GET /payroll/reports/cost` (domain API)

---

## Overview

Consolidated payroll cost breakdown by cost center and department for a given period. Enables Finance to reconcile payroll expense against GL accounts and HR to monitor headcount costs.

---

## Data Sources

| Source Table | Usage |
|-------------|-------|
| `payroll_result` | gross_amount, si_employee_total, pit_amount, net_amount per worker per period |
| `compensation_snapshot` | cost_center_id, pay_group_id for cost allocation |
| `payroll_run` | run_type, run_mode (filter for PRODUCTION + APPROVED/LOCKED only) |
| `pay_group` | pay_group_name, legal_entity_id |

---

## Metrics

| Metric | Definition | Aggregation |
|--------|-----------|-------------|
| Total Gross | Sum of payroll_result.gross_amount | SUM per dimension |
| Employer BHXH | gross_amount × 17.5% (BR-021) — from statutory_rule | Computed |
| Employer BHYT | gross_amount × 3% (BR-023) | Computed |
| Employer BHTN | gross_amount × 1% (BR-025) | Computed |
| Total Employer SI | Sum of employer BHXH + BHYT + BHTN | SUM |
| Total Payroll Cost | Total Gross + Total Employer SI | SUM |
| Worker Count | Distinct working_relationship_id in run | COUNT |
| Avg Cost per Worker | Total Payroll Cost / Worker Count | Computed |

---

## Dimensions

| Dimension | Values |
|-----------|--------|
| Legal Entity | legal_entity_id (from context switcher) |
| Period | pay_period (month/year selector) |
| Cost Center | cost_center_id from compensation_snapshot |
| Pay Group | pay_group_id |
| Pay Method | pay_method from compensation_snapshot |

---

## Filters

- **Legal Entity**: pre-filled from global context switcher; CE_VIEWER can select multiple
- **Period**: month/year picker; range up to 12 months
- **Pay Group**: multi-select
- **Cost Center**: multi-select (tree select if hierarchical)
- **Pay Method**: multi-select

---

## Visualization

1. **Summary Cards** (top): Total Gross | Total Employer SI | Total Payroll Cost | Worker Count
2. **Bar Chart**: Payroll Cost by Cost Center (stacked: Gross + Employer SI)
3. **Detail Table**: one row per cost_center × pay_group; columns: Cost Center, Pay Group, Worker Count, Total Gross, Employer SI, Total Cost, Avg Cost/Worker
4. **Period Comparison** (optional toggle): side-by-side current vs previous period

## Business Rules Applied

| Rule | Description |
|------|-------------|
| BR-021 | Employer BHXH rate: 17.5% of SI basis |
| BR-023 | Employer BHYT rate: 3% of SI basis |
| BR-025 | Employer BHTN rate: 1% of SI basis |
| BR-100 | Data scoped to legal_entity_id |
