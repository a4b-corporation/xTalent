# User Stories: [Module Name]

> **Instructions**: Document user stories in this structured format.
> AI will extract entities, workflows, and business rules from these stories.
> The more detail you provide, the better the ontology output.

## Story Format

Each story follows:
- **As a** [role]
- **I want to** [action/capability]
- **So that** [benefit/value]

Plus:
- **Acceptance Criteria** (specific, testable conditions)
- **Notes** (additional context, edge cases)
- **Priority** (Must/Should/Could/Won't)
