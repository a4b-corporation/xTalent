# Form Layout Standards

> Tiêu chuẩn cho layout các form nhập liệu

---

## 1. Form Layout Types

### 1.1 Single Column (Default)

```
┌─────────────────────────────────────────────────────────────────────┐
│ Field 1                                                             │
│ [___________________________________________________]              │
│                                                                     │
│ Field 2                                                             │
│ [___________________________________________________]              │
│                                                                     │
│ Field 3                                                             │
│ [___________________________________________________]              │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.2 Two Column

```
┌─────────────────────────────────────────────────────────────────────┐
│ Field 1                           Field 2                           │
│ [_______________________]        [_______________________]         │
│                                                                     │
│ Field 3                           Field 4                           │
│ [_______________________]        [_______________________]         │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.3 Mixed Widths

```
┌─────────────────────────────────────────────────────────────────────┐
│ Full Width Field                                                    │
│ [___________________________________________________]              │
│                                                                     │
│ Field 2 (50%)                     Field 3 (50%)                     │
│ [_______________________]        [_______________________]         │
│                                                                     │
│ Short (25%)    Medium (25%)       Long (50%)                        │
│ [_______]     [_______]          [_______________________]         │
└─────────────────────────────────────────────────────────────────────┘
```

## 2. Form Sections

```
┌─────────────────────────────────────────────────────────────────────┐
│ Section Title                                                 [−]  │
│ ─────────────────────────────────────────────────────────────────  │
│                                                                     │
│ [Form fields...]                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 3. Label Positions

| Position | Usage |
|----------|-------|
| **Top** (Default) | Most forms, mobile-friendly |
| **Left** (Inline) | Compact forms, desktop |
| **Floating** | Login/signup forms |

---

## 4. Field Spacing

| Element | Spacing |
|---------|---------|
| Between fields | 16-24px |
| Label to input | 4-8px |
| Between sections | 32px |
| Form padding | 24px |

---

## 5. Quy Định

| Mã | Quy định | Priority |
|----|----------|----------|
| FL-01 | Mobile = single column | P0 |
| FL-02 | Required indicator (*) | P0 |
| FL-03 | Error messages below field | P0 |
| FL-04 | Tab order logical | P0 |
| FL-05 | Group related fields | P1 |
| FL-06 | Max 70 chars input width | P1 |
